Privacy Policy

WHOOP Debrief — Last updated July 2026

What this app is

WHOOP Debrief is a personal data pipeline built and used solely by Edward Lai. It is not a product, service, or commercial application. There are no other users.

What data is collected

The app reads Edward's own WHOOP data via the WHOOP API: daily recovery scores, sleep records, strain cycles, and workout summaries. No data from any other WHOOP account is accessed or collected.

Where data goes

Fetched data is written to Edward's private Google BigQuery project. It is not published, transmitted to any third party, or exposed publicly.

Data sharing and selling

No data is sold, shared, or disclosed to any third party. The pipeline runs on GitHub Actions with secrets stored as encrypted environment variables.

Data retention

Data lives in a private BigQuery dataset and is retained at Edward's discretion. There is no automatic deletion policy because there are no external users.

Contact

Questions? Reach out via the contact page on this site.

This policy applies only to the WHOOP Debrief personal project. It does not represent a commercial privacy policy.
